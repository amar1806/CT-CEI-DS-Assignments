# CT-CEI-DS-Assignments

A structured repository for weekly assignment submissions completed during the **Celebal Excellence Internship (CEI) – Data Science Program** at Celebal Technologies.

This repository contains hands-on implementations and practice tasks related to **Python, Statistics, NumPy, Pandas, Linear Algebra, Data Visualization, and Machine Learning fundamentals**. Each week’s assignments are uploaded regularly to track learning progress and strengthen practical Data Science skills.
